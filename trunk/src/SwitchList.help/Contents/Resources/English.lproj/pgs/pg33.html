User Interface: Style Tab

The Style tab in the layout window controls which switchlist style you use for your layout. You'll be able to choose between any of the standard Switchlist document styles, as well as any custom switchlists you've created and installed on your computer.

To choose a different switchlist style, use the pull-down menu on the left to choose a switchlist by name. The text below the button will change to explain a bit about that switchlist style, and the box to the right will show an example of the switchlist. Click on the example to see a full-size version.

The table below the pull-down allows you to customize some text in the switchlist. To customize, just double-click on a value in the right column, and type your preferred text. Your text is rendered into the switchlist as if it were a web page, so many of the formatting tricks that work for web pages also work in switchlists. "<br>", for example will insert a line break; "<b>My Name<b>" will draw My Name in bold text.

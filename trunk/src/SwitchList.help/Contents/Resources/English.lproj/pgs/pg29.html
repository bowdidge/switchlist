User Interface: Preferences Window

Preferences that apply to all layouts are set from the Preferences pane. You can set these by choosing the "Preferences..." menu item in the SwitchList menu and changing settings in the pane.

There are two settings in Preferences: it controls the kind of switch list that is displayed, and also controls whether the "Switchlists on the iPad" feature is enabled.

To change the kind of switchlists displayed, pull down the menu and select the switchlist you prefer. You'll be able to choose between any of the standard Switchlist document styles, as well as any custom switchlists you've created and installed on your computer.

Customization: Creating Custom SwitchList Styles

As cute as the default SwitchList styles might be, you might find the standard ones not quite right. Maybe the font sizes are too small, your operators need additional hints or details, or you'd like to match the look of your favorite railroad's paperwork.

Luckily, SwitchList is customizable; by creating an HTML file that describes your switchlist's appearance and dropping it in a special location on your computer, your switchlist style will appear as a choice in the Style tab.

Full details are available at the SwitchList development site over on code.google.com, but here's a quick rundown.

  • First, download the set of alternate switchlist styles to use those as a starting point for your custom switchlist. Download the "Custom Switchlist Templates.zip" file from SwitchList download site, and poke around at the examples. The Thomas the Tank Engine switchlist example shows what can be done with a small bit of work.

  • Second, create your own switchlist from one of these starting points. Create an HTML file (the same thing inside every web page) describing your preferred switchlist, color and background, and the like. You'll place special placeholders in your file to indicate where freight cars and other information gets placed. Place the switchlist.html in a directory with the name you'd like to see in SwitchList's switchlist style list.

  • Third, copy your custom switchlist into SwitchList's special directory. You can do this in two ways: via the "Import Template..." menu item in the File menu, or by copying the file yourself. Using "Import Template..." is the easiest way - just select the directory containing your switchlist.html template, and hit OK. The template will be copied into place. Once the template is copied in, start using it by selecting the switchlist style with the name of the directory.

If you want to copy the template directories in yourself, note that the special directory can be in one of two places depending on how you got SwitchList. If you installed SwitchList via the Mac App Store, the templates are placed in the directory: 

~/Library/Containers/com.blogspot.vasonabranch.SwitchListApp/Data/Library/Application Support/SwitchList

If you installed SwitchList by downloading it from www.vasonabranch.com, or copying it from a friend's machine, the templates are placed in the directory: 

~/Library/Application Support/SwitchList

Full details are available SwitchList development site. Note that you only need to copy the files into place once; as you experiment with switchlist styles, you can just modify the copies of your switchlist style in the special directory, and see the changes immediately within SwitchList. You can still use the "Import Template..." on an already-installed switchlist template, though it will replace the previous copy.

Now, run SwitchList and you'll see your custom switchlist in the list of styles. It will display, print, and show up in the iPad / web interface just like any other switchlist!

These same instructions hold if you want to just use a custom switchlist style - use the "Import Template..." menu to copy the switchlist style into SwitchList, and choose the new style as your preferred switchlist format.

Again, the SwitchList development site has details on how to create your own switchlists and other reports, and customize them to match your needs. Also check out the SwitchList mailing list for advice on writing your own switchlist style.

SwitchList also allows you to customize many of the standard reports - yard report, car report, etc. - to match your switchlist paperwork. Check out the SwitchList development site for more details.
